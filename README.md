# Base64 Encoding and Decoding in C++

A simple C++ implementation of Base64 encoding and decoding, written from scratch without any external libraries.

## Features

- Encodes and decodes Base64 strings.
- Self-contained: no external dependencies or libraries required.
- Includes easy-to-use `Encode` and `Decode` methods.

## Usage

To use this Base64 utility, include `base64.h` in your project and call `Base64::Encode` or `Base64::Decode` as shown in `main.cpp`.
